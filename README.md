# 강의 진행 도움툴 (iPad PWA)

강사가 iPad를 손에 들고 강의를 운영하기 위한 **웹 기반 스탠드얼론(오프라인) 진행 보조 도구**입니다.
빌드 과정 없는 순수 정적 PWA로, 인터넷 없이 동작하고 홈 화면에 앱처럼 추가할 수 있습니다.

기본 콘텐츠로 **"AI 가상모델 제작 워크숍"**(1일 6시간) 커리큘럼이 채워져 있으며,
인앱 편집과 JSON Import/Export로 **다른 강의에도 재사용**할 수 있습니다.

## 핵심 기능
- **오늘(홈)**: 현재 교시 + 대형 카운트다운 타이머(green→amber→red 색 전환), 하루 진행률, 교시 빠른 이동
- **타임라인**: 교시별 시간/제목/상태(완료·진행중·예정), 점심시간 표시
- **교시 상세**: 학습 목표, 발표자 메모(마크다운), 실습 체크, 관련 치트시트 바로가기, 교시 타이머
- **치트시트**: 툴별 탭(ChatGPT/Midjourney/ComfyUI/Photoshop), 프롬프트 **한 번 탭 복사**, 검색
- **평가 체크리스트**: 평가자 체크 + 교시별 실습 체크, 진행률 (localStorage 영속)
- **설정/편집**: 강의 내용 편집, 테마/글자 크기, 여러 강의 프로필 전환, JSON 가져오기·내보내기

## 로컬 실행
정적 파일이라 아무 정적 서버로 열면 됩니다.
```bash
python3 -m http.server 8080
# 브라우저에서 http://localhost:8080
```
> 모듈/서비스워커/클립보드는 `http://localhost` 또는 HTTPS 환경에서 동작합니다. (file:// 직접 열기는 권장하지 않음)

## iPad에서 사용 (홈 화면 추가)
1. 배포 URL을 **Safari**로 엽니다.
2. 공유 버튼 → **홈 화면에 추가**.
3. 추가된 아이콘으로 실행하면 전체화면 앱처럼 동작하고, 한 번 연 뒤에는 오프라인에서도 사용 가능합니다.

## 배포 (GitHub Pages)
모든 경로가 상대경로라 별도 빌드 없이 그대로 호스팅됩니다.
저장소 **Settings → Pages**에서 브랜치를 선택해 게시하면 됩니다.

## 데이터 / 재사용
- 모든 강의 내용은 하나의 **강의 프로필(JSON)** 로 표현됩니다. (`data/default-lecture.js` 가 기본 시드)
- 설정 화면에서 현재 강의를 JSON으로 **내보내기**(파일/클립보드) 하고, 다른 기기·다른 강의에서 **가져오기** 할 수 있습니다.
- 데이터와 설정은 브라우저 `localStorage` 에 저장됩니다. ("기본값 복원"으로 시드 내용으로 되돌릴 수 있음)

## 구조
```
index.html               앱 셸
manifest.webmanifest     PWA 매니페스트
sw.js                    서비스워커(오프라인 캐시)
css/styles.css           스타일(다크/라이트·반응형)
js/app.js                라우터/부트스트랩
js/state.js              상태·프로필·import/export
js/timer.js              교시 타이머
js/markdown.js           경량 마크다운 렌더러
js/util.js               공용 헬퍼
js/views/*.js            화면별 렌더러
data/default-lecture.js  기본 강의 시드
icons/                   PWA 아이콘
```
